# XPS_UPS (UPS IBW Processor)

用于处理 UPS 的 `.ibw` 数据：预览绘图、计算功函数、导出 CSV/PNG。

## 运行（Windows）

- **方式 B（对方电脑已安装 Python / Anaconda）**
  - 进入本项目文件夹，双击 `运行.bat`
  - 或在该文件夹打开终端运行：`python XPS_UPS.py`

- **方式 A（推荐：解压即用、对方无需安装 Python）**
  - 参考 `README_发给别人怎么跑.md`

## 依赖

```text
numpy
matplotlib
igor2
pandas
```

安装方式：

```bash
pip install -r requirements.txt
```
